# AI 수익형 블로그 완성 가이드 (I Am Not AI)

## 이 스킬이 하는 일

AI 툴을 활용해 **네이버 수익형 블로그 글을 처음부터 끝까지** 완성한다.
단순한 윤문이 아니라, 리서치부터 제휴링크 삽입, 이미지 배치, 네이버 최적화까지 전 과정을 커버한다.

---

## 사용자에게 먼저 확인할 것

스킬 시작 전, 다음을 확인한다:
- **주제/키워드**: 어떤 주제로 쓸 건지
- **AI 초안 유무**: 이미 쓴 글이 있는지, 처음부터 써야 하는지
- **제휴링크 유무**: 세시간전에서 연결할 상품이 있는지
- **이미지 방향**: AI 생성 이미지 원하는지, Unsplash 무료 이미지 원하는지

---

```
1단계: NotebookLM 리서치
2단계: AI 초안 작성
3단계: 이미지 선택 & 배치
4단계: 세시간전 제휴링크 삽입
5단계: 윤문 (AI → 사람 말투)

```

## 1단계: NotebookLM으로 리서치하기

NotebookLM은 자료를 넣으면 AI가 분석해주는 구글 도구다. 블로그 주제를 깊게 파고들 때 유용하다.

### NotebookLM 사용법 (처음이라면)

1. **notebook.google.com** 접속 → 구글 계정으로 로그인
2. **"새 노트북"** 클릭
3. 소스 추가하는 방법:
   - URL 붙여넣기 (참고할 웹페이지 주소)
   - PDF 파일 업로드
   - 직접 텍스트 붙여넣기
4. 소스를 2~5개 추가한 후 채팅창에 질문

### NotebookLM에 물어볼 질문 예시

주제가 "아이폰 배터리 교체"라면:
```
- "이 자료들 요약해줘"
- "독자가 가장 궁금해할 질문 5가지 뭐야?"
- "경쟁 블로그와 차별화할 수 있는 포인트가 뭐야?"
- "이 주제로 블로그 목차 짜줘"
```

### NotebookLM 없이 리서치할 때

사용자가 NotebookLM을 건너뛰고 싶다면:
- 네이버 검색에서 상위 5개 블로그 분석
- "이 주제로 독자가 진짜 궁금해하는 것"을 Claude에게 직접 물어보기
- 연관 검색어로 롱테일 키워드 찾기

## 2단계: AI 초안 작성

### 이미 AI로 쓴 글이 있을 때

3단계로 건너뛴다.

### 처음부터 작성할 때

다음 구조로 초안을 만든다:

**네이버 블로그 기본 구조:**
```
제목: [키워드 포함] + [클릭 유발 요소]
  예) "아이폰 배터리 교체 비용, 공식 vs 사설 직접 비교해봤어요"

도입부 (3~5줄):
  - 독자 공감 or 경험담으로 시작
  - 이 글에서 얻을 수 있는 것 한 줄로 예고

본문 (소제목 3~5개):
  - 소제목마다 이미지 1장
  - 각 소제목 아래 200~300자

마무리 (3~5줄):
  - 핵심 요약
  - 독자 행동 유도 (댓글, 공유, 제품 링크 클릭)
```

### 각 AI 툴 활용 방법

| AI 툴 | 잘하는 것 | 추천 사용 방법 |
|-------|----------|--------------|
| NotebookLM | 자료 분석, 요약 | 리서치, 목차 구성 |
| Gemini | 최신 정보 검색 | 가격/트렌드 조사 |
| ChatGPT | 구조적인 글쓰기 | 초안 작성 |
| Genspark | 빠른 콘텐츠 생성 | 초안 초고 |
| Claude | 자연스러운 윤문 | 최종 다듬기 |

## 3단계: 이미지 선택 & 배치

### Unsplash 무료 이미지 찾기

1. **unsplash.com** 접속
2. 영어로 검색 (한국어보다 결과 풍부)
   - "iPhone battery" / "diet food" / "smartphone"
3. 다운로드 후 사용 (저작권 무료, 출처 표기 선택사항)

### AI 이미지 생성할 때

Claude나 다른 AI에게 이렇게 요청:
```
"[주제]와 어울리는 블로그 섬네일 이미지를 생성해줘.
스타일: 밝고 깔끔한 느낌, 한국 블로그 감성
텍스트 없이 이미지만"
```

### 이미지 배치 원칙

- **섬네일**: 제목 바로 아래 (가장 중요)
- **본문 중간**: 소제목마다 1장 (독자 시선 유지)
- **마무리 전**: 제품 이미지 또는 제휴 상품 관련 이미지

글 안에서 이미지 위치는 `[이미지: 설명]` 형식으로 표시한다.

## 4단계: 세시간전 제휴링크 자연스럽게 삽입

### 제휴링크를 녹이는 핵심 원칙

광고처럼 보이면 독자가 떠난다. **정보의 흐름 속에 자연스럽게** 끼워넣어야 한다.

### 삽입 위치 3곳

**① 본문 중간 — 추천 형식**
```
저는 요즘 이걸 쓰고 있는데 솔직히 꽤 만족스러워요.
→ [상품명 + 제휴링크]
가격 대비 괜찮은 것 같더라고요.
```

**② 비교 섹션 — 정보형**
```
비슷한 제품들 찾아봤는데, 가성비로는 이게 제일 낫더라고요.
→ [상품명 + 제휴링크] (현재 가격 확인하기)
```

**③ 마무리 — 행동 유도형**
```
직접 써보고 싶은 분들은 아래 링크에서 확인해보세요.
현재 할인 중이라 지금이 타이밍인 것 같아요.
→ [상품명 + 제휴링크]
```

### 절대 하지 말아야 할 것

- "※ 이 포스팅은 제휴마케팅이 포함되어 있습니다" 문구를 글 시작에 크게 넣기 (흐름 방해)
- 링크를 3개 이상 한꺼번에 나열하기
- "구매하세요!", "지금 바로 클릭!" 같은 직접적인 광고 문구

## 5단계: 윤문 — AI 글을 사람 글로

### AI 글 패턴 감지 & 제거

**제거 대상 표현:**
- "~라고 할 수 있습니다" → "~예요/이에요"
- "이뿐만 아니라", "더불어", "또한" 연속 사용 → 삭제하거나 다른 방식으로
- "지금까지 ~에 대해 알아보았습니다" → 자연스러운 마무리로 교체
- 첫째, 둘째, 셋째 나열 → 소제목으로 분리

**추가할 인간적 요소:**
- 짧은 감탄문: "이거 진짜 신기하더라고요", "생각보다 훨씬 낫더라고요"
- 개인 경험처럼 느껴지는 1인칭 표현
- 길고 짧은 문장 리듬감 있게 섞기
- 독자에게 말 거는 표현: "아시죠?", "해보셨나요?"

### 윤문 체크리스트

글 완성 전 다음을 확인한다:
- [ ] 첫 문장이 독자를 잡아당기는가?
- [ ] "~습니다" 말투가 과하게 반복되지 않는가?
- [ ] 소제목이 궁금증을 유발하는가?
- [ ] 제휴링크가 광고처럼 보이지 않는가?
- [ ] 마무리가 자연스럽게 끝나는가?

## 6단계: 네이버 블로그 최적화 포맷

### 최종 출력 형식

```
📌 [제목]
(키워드 포함, 클릭 유발)

[이미지: 섬네일 - 주제 관련 이미지]

[도입부]
(독자 공감 or 경험담, 3~5줄)

---

## [소제목 1]
[이미지: 관련 이미지]
(본문 200~300자)

## [소제목 2]
[이미지: 관련 이미지]
(본문 200~300자)
→ 제휴링크 삽입 위치 표시

## [소제목 3]
[이미지: 관련 이미지]
(본문 200~300자)

---

[마무리]
(핵심 요약 + 독자 행동 유도)
→ 제휴링크 마무리 삽입

#해시태그1 #해시태그2 #해시태그3 #해시태그4 #해시태그5
```

### 네이버 SEO 팁

- **제목**: 30자 이내, 핵심 키워드 앞쪽에
- **본문 길이**: 1,500자 이상 (너무 짧으면 노출 불리)
- **해시태그**: 5~10개, 키워드 + 관련어 혼합
- **이미지**: 최소 3장 이상, 직접 촬영 or 편집한 이미지 우대
- **발행 시간**: 오전 7~9시, 오후 12~1시, 오후 9~11시 노출 유리
